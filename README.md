# 3x3 Output Stationary Systolic Array for GEMM

## About

**ECE 204: Digital Logic Design** Final Group Project by Bryce Dreyer and Lucas Wilson, Oregon State University. 3x3 Systolic Array Designed for General Matrix Multiply (GEMM) operations, taking two 3x3 matrices as input and returning their matrix product. This project is a snapshot into systolic arrays, a specialized computer architecture comprised of interconnected processing elements. Scaled larger arrays are applied intensive deep learning workloads (such as Google's Tensor Processing Unit), Convolutional Neural Networks, Digital Signal Processing, and much more.  

Design verification via ModelSim. Project targets the Terasic DE10 Lite FPGA Board (class materials).
